# Autonomous Trading AI (Simulation)

This project demonstrates a **self-evolving autonomous AI trader** that:
- Observes market prices
- Makes Buy / Sell / Hold decisions
- Learns from profit & loss
- Detects market changes (concept drift)
- Adapts automatically without retraining

⚠️ Simulation only — no real money trading.
